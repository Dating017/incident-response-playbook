# Incident Response Playbook

## Overview
A professional Incident Response (IR) Playbook developed for Golden Crunch Biscuits Ltd., 
covering three real-world cyberattack scenarios. Built as part of the TechCrush 
Cybersecurity Capstone Project (Group 30).

## Scenarios Covered
- **Business Email Compromise (BEC)** — Supplier impersonation and payment fraud
- **Ransomware (NetWalker)** — Endpoint infection, lateral movement, and recovery
- **DDoS Attack** — Volumetric and application-layer flood mitigation

## IR Phases Covered
- Detection & Identification
- Containment
- Eradication
- Recovery
- Post-Incident Review (PIR)

| Tool | Purpose |

| Mimecast / Microsoft Defender | Email Security Gateway |
| Cisco Umbrella | DNS Filtering |
| VirusTotal / MISP | Threat Intelligence |
| Cloudflare / AWS Shield | DDoS Scrubbing |

## Key Metrics (KPIs)
| Metric | Target |
|--------|--------|
| MTTD (Mean Time to Detect) | Under 24 hours |
| MTTR (Mean Time to Respond) | Under 5 hours |
| Legitimate Traffic False Positive Rate | Under 1% |
